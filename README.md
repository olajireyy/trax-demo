# Trax: Personalized Music Insights (Portfolio Demo)

Trax is a premium music analytics dashboard built with **React** and **Vite**. It features a modern, glassmorphic UI designed to provide users with deep insights into their listening habits, social interactions, and AI-generated "taste roasts."

> [!NOTE]
> This is a **Standalone Portfolio Demo**. It uses a built-in Mock API to simulate a full backend and Spotify integration, making it perfect for immediate review without any external dependencies.

## 🚀 Key Features

*   **Cosmic Insights**: Multi-timeframe analytics (Today, 7D, 4W, 6M) with interactive 24-hour listening trends.
*   **Social Hub**: A fully functional "Mutuals" system with real-time activity feeds and direct messaging.
*   **Taste DNA**: AI-powered analysis of musical uniqueness and compatibility with friends.
*   **Daily Roast**: A playful, AI-driven critique of your recent listening "sins."
*   **Standalone Mode**: 100% mocked backend infrastructure for seamless, zero-config hosting.

## 🛠️ Tech Stack

*   **Frontend**: React 19, Vite 7
*   **Styling**: Tailwind CSS 4, Framer Motion (Animations)
*   **Icons**: Lucide React
*   **API Layer**: Custom Mock Adapter (Standalone)

## 📦 Getting Started

1.  **Clone & Install**:
    ```bash
    npm install
    ```
2.  **Run Locally**:
    ```bash
    npm run dev
    ```
3.  **Build for Production**:
    ```bash
    npm run build
    ```

## 🌐 Hosting

This project is optimized for static hosting on **Vercel**, **Netlify**, or **GitHub Pages**. Simply build the project and deploy the `dist/` folder.
